# jpmj_games_record

麻雀記録帳
JPMJ Games Record

麻雀ゲームの試合状況を記録するアプリです。
An application used to take record during Japanese Mahjong games.

## Getting Started

### Prerequisite

Flutter version: 3.0.5+

### Install Dependencies

```
flutter pub get
```

### Run app in CLI

```
flutter run -d <DEVICE_NAME>
```
